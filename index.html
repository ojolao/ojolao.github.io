About Me

My name is Tayo Jolaosho. I am a software engineer with a background in anthropology. Fluent in multiple JS frameworks, I am interested in working on innovative technical projects.

As a child, I was always called upon by neighbors and family members to repair electronics and program devices, including TVs and VCRs, to function at the right settings. These requests allowed me to cultivate an intuitive grasp of how things can work together that has shaped my ongoing passion for software engineering. I love the transformational capacity that programming affords—being able to move a system through integrative processes that fundamentally shift amorphous or latent ideas into elegant, well-considered implementations. PROJECTS

Friend.ly

Friend.ly is an iOS app that allows users to create events and invite friends, who can propose alternative activities and vote on their preferences. It was built using React Native and Redux with Express for Node.js on the backend and MySQL for a relational database. I designed a low-latency live activity feed system by integrating Socket.io with Node/Express and MySQL for bi-directional dynamic messaging and tailored client-side notifications. Integrating the Mailgun API with Node/Express enabled me to develop an expeditious and scalable email customization feature that engages users off the app. Furthermore, I created modular front-end components with React Native for enhanced rendering speed and extensibility.

Diff

Diff is a responsively designed web app that converts photos of receipts into digital, interactive representations through an OCR algorithm and leveraging of Google's Vision API. The app enables simple and quick calculation of monetary contribution owed by those splitting expenses. Working across the stack, I developed a performant customized email notification system by using the Mailgun API with Node/Express. On the front end, I refactored legacy React code integrating views and view logic into existing components to enhance the app’s intuitive and interactive interface.

Viz.ly

Viz.ly is a data visualizer to help users understand personal trends in their photo collections. It leverages Google’s Vision API for insightful image analysis, which is then powerfully rendered using interactive D3.js charts. I implemented an authentication process using PassportJS and Facebook Login for a simple and secure workflow. I also constructed database schema using MongoDB and Mongoose for efficient data storage and retrieval.

Follow Up

Follow Up is a single page web app for prioritizable task management. I built the app using HTML5 to create draggable React components. I also composed a RESTful API using Node with Express and MySQL for data persistence and optimized performance.
